## 세션 시퀀스와 재심사 타이밍

1. **S1** — 웹진 인프라(/guide 라우트·프리렌더·사이트맵·홈 스트립) + 기사 #1·#2 + E-E-A-T 블록 → 배포, 서치콘솔·네이버 sitemap 재제출 + 색인 요청
2. **S2** — 기사 #3·#4 + 은행 허브 + 비교 페이지 2 보강
3. **S3** — 은행 가이드 배치1(7) + 기사 #5(링크 메시 동시 발행)
4. **S4** — 기사 #6~#8
5. **S5** — 은행 가이드 배치2(7) + 기사 #9~#12
6. **S6** — 재심사 요청. 달력 기준이 아니라 아래 **측정 가능 게이트 전부 충족** 시점(경험상 S3 착지 후 2~4주):
   - [ ] 서치콘솔에서 /guide/* 및 은행 상세 페이지 과반이 "색인 생성됨"
   - [ ] 수동 조치(Manual Actions) 0건
   - [ ] 전 페이지 프리렌더 ≥~1,000자, 고아 URL 없음
   - [ ] 바이라인·출처 링크가 전 기사에 라이브
   - (수정 없이 즉시 재요청 금지 — 반복 거절 이력은 불리)
