# 청년도약계좌 ↔ 청년미래적금 갈아타기 계산 Spec (확정)

> 기준일 2026-06-23 / 출처: 금융위(86767·87005·87106·87158), korea.kr 정책브리핑, 은행연합회 소비자포털, 토스뱅크, FSC 83782(기여금확대), 하나은행 상품설명. deep-research(고신뢰) + 보강조사.
> 모든 금리·일정은 가입 전 은행·서민금융진흥원 재확인 필요(면책).

## 1. 확정 데이터

### 청년미래적금 (2026 신규)
- 만기 **3년(36개월)**, 월 최대 **50만원** 자유적립, **3년 고정금리**, 이자 **비과세**.
- 기본금리 **연 5%**(전 기관 동일) + 기관우대 **3%p**(국민·신한·하나·우리·NH농협·IBK기업·우체국) 또는 **2%p**(수협·iM·부산·광주·전북·경남·카카오) + 공통우대 **저소득 0.5%p**(총급여 3,600만↓) + **재무상담 0.2%p**.
  → 명목 최고 **8%**(3%p그룹) / **7%**(2%p그룹). (※ 케이뱅크 미참여, 토스뱅크 12월 별도출시)
- 정부기여금: **월 납입액의 6%(일반형) / 12%(우대형)**, 매월 매칭. 월50만 시 일반 3만·우대 6만 → 3년 총 일반 108만·우대 216만.
- 우대형(12%) 자격: **중소기업 신규취업자** = 가입신청일 기준 **전년도(2025.1~12) 최초취업 + 현재 중소기업 재직**. 생애최초 아니어도 해당기업 취업 전 고용보험 합산 **1년 미만**이면 신규취업 인정.
- 가입요건: 만 **19~34세**(병역 최대 6년 미산입), 총급여 **7,500만 이하** 가입가능 / **6,000만 초과~7,500만**은 기여금 0·비과세만 / **6,000만 이하** 기여금 지급.
- '최대 연 19.4%'는 명목금리 아님 = 금리(8% 가정)+기여금+비과세 **실질효과(단리환산)**. 일반형 13.2~14.4%, 우대형 18.2~19.4%.
- 일정: 신청 6/22~7/3(첫주 출생연도 5부제), 심사 7/6~24, 계좌개설 7/27~8/7.

### 청년도약계좌 (현행)
- 만기 **5년(60개월)**, 월 최대 **70만원**, 이자 **비과세**.
- 금리: 기본 **최초 3년 고정 + 이후 1년마다 변동**. 보도 '4.5%→3.0%'는 3년고정 종료 후 변동분 하락. (사용자별 상이 → 입력)
- 정부기여금(2025.1 확대, 매칭한도 전구간 월70만 통일): 소득구간별 월 최대 기여금
  | 총급여 | 산식(월, 만원) | 월 최대 기여금 |
  |---|---|---|
  | ≤2,400 | min(m,40)×6.0% + (min(m,70)−40)⁺×3.0% | 3.3만 |
  | 2,400~3,600 | min(m,50)×4.6% + (min(m,70)−50)⁺×3.0% | 2.9만 |
  | 3,600~4,800 | min(m,60)×3.7% + (min(m,70)−60)⁺×3.0% | 2.52만 |
  | 4,800~6,000 | min(m,70)×3.0% | 2.1만 |
  | 6,000~7,500 | 0 (비과세만) | 0 |
  ( m=월납입액(만원), (x)⁺=max(0,x) )

### 갈아타기 (가장 중요)
- 전환은 **2026.6 최초 가입기간(신청 6/22~7/3, 계좌개설 ~8/7)에만** 허용. 일회성.
- **순서 필수**: ①미래적금 신청 → ②승인 → ③미래적금 계좌개설 → ④도약계좌 **특별중도해지** → ⑤미래적금 납입. 역순(도약 먼저 해지)이면 혜택 소멸.
- **특별중도해지 = 기존 납입분의 정부기여금·비과세 손실 없이 환급금에 보존**(일반 중도해지는 둘 다 상실).
- **해지수령금의 미래적금 일시납입 불가** → 도약 원금·납입기간은 승계 안 됨(현금 환급). 신용점수 가점은 도약 기간·납입액 합산.
- 동시 보유 불가.

## 2. 계산 모델

### 적금 이자(단리 자유적립, 비과세)
월 정액 m(원), n개월, 연리 r: `세전이자 I = m × r × n(n+1)/2 / 12`, 비과세이므로 세후=세전.

### 핵심 통찰 — 과거분은 양쪽 모두 보존
유지: 과거 납입분이 도약 만기까지 계속 적립·이자. 전환: 과거분이 특별중도해지로 **손실 없이 현금 환급**(기여금·비과세 포함).
→ 의사결정은 **"앞으로의 돈"** 비교가 본질. 과거분의 차이는 (a) 전환 시 환급현금의 재운용 수익률, (b) 유지 시 도약 잔여기간 이자.

### 경로별 미래가치 (지금=T, 도약 경과 e개월, 잔여 R=60−e)
- **유지(STAY)**: 도약 만기수령 = 60개월 납입원금 + 60개월 기여금누계 + 도약이자(원금·60개월·r_도약). 추가투입 가능 ≤70만/월.
- **전환(SWITCH)**: 도약 특별중도해지 환급금(과거원금+과거기여금+과거이자, 손실0) + 미래적금 만기수령(36개월 원금 + 기여금(6/12%)×36 + 미래이자(원금·36개월·r_미래)). 미래 투입 ≤50만/월.
  - **환급현금 재운용은 모델링하지 않음(구현 결정).** 환급금은 손실 없이 보존되는 '현재 현금'으로 별도 표기만 하고(코드: `compute().refund`, `ComparePanels`에 분리 표시), 가상의 운용금리 r_o로 굴리지 않는다. 사용자가 답할 수 없는 가정값(r_o)을 만들지 않기 위함이며, 의사결정은 아래 §"핵심 통찰"대로 **앞으로의 돈(diff3yr)** 비교가 본질이라 r_o는 2차 요인이다. (별개로 `investReturn`은 "적금 대신 투자할까?" 비교용 입력으로, 환급금 재운용과 무관하다 — `lib/invest.ts`.)

### 비교·추천 지표
1. **연 실질효과율(단리환산)**: 도약유지 vs 미래적금 각각. 미래 우대형 ≈ 금리+12%, 일반형 ≈ 금리+6%; 도약 ≈ r_도약 + (기여금/납입액).
2. **순이익 비교**: 동일 월납입 가정(min(70,50) 또는 사용자값)에서 [지금→미래만기 36개월] 구간 순수익(이자+기여금) 비교(코드 `diff3yr`). 환급현금 재운용분은 모델에서 제외(위 SWITCH 참조).
3. **유동성**: 도약 잔여 R개월 락 vs 미래 36개월.

## 3. 추천 분기 규칙(확정)
- **우대형(12%) 자격** → 강한 전환 신호(기여금률 2배 + 단기 + 고금리). 거의 항상 전환 유리.
- **일반형 + 도약 잔여 김(초기)** → 전환 우세(미래 6%기여+고금리+3년, 5년 락 회피). 단 도약 저소득(≤2,400, 기여 3.3만≈월납70만의 4.7%)이고 70만 꽉 채워 납입 중이면 도약 기여 총액이 커 경합.
- **도약 만기 임박(R 작음, 예 ≤12개월)** → 유지하여 마무리 권장(곧 만기, 전환은 새 3년 시작·실익 적음).
- **6,000만 초과 소득(도약 기여금 0)** → 비과세만 받는 중이므로 미래적금(기여금+고금리) 전환 강하게 유리(우대형이면 더).
- 출력: "유지 시 총수령 X / 전환 시 총수령 Y + 도약 환급금(별도 현금) / 같은 돈 3년 순이익 차액 Z" + 한 줄 결론 + 근거(요인별) + 유동성/주의(순서·일정) 경고.

## 4. 입력 변수
공통: 만나이, 총급여(소득구간 자동), 목적(목돈/유동성).
도약(보유): 가입연월→경과개월, 월납입액(≤70), 적용금리(기본 4.5%).
미래(전환): 유형(일반/우대)+중소기업신규취업 체크, 월납입액(≤50), 은행(전 14개 기관), 우대조건 충족 토글.
별도 비교축: 투자 가정수익률 `investReturn`(기본7%) — "적금 대신 투자할까?" 패널용이며 환급금 재운용과 무관.

## 5. 시나리오 분기표(축)
- 유형(일반형/우대형) × 소득구간(5) × 도약경과(예 12·24·36·48개월) → 셀=추천(유지/전환)+순이익차.

## 6. 테스트(경계·시나리오)
- 경과 0/59개월, 납입 0/상한(70·50), 소득구간 경계, 일반↔우대, 은행 최고↔최저.
- 적금이자 공식 검증(FSC 예시 ±오차 설명), 기여금 산식 5구간 검증, 추천 분기 안정성.
