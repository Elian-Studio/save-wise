## 공통 사실 (전 은행 동일, 정부 표준)

- 기본금리 5.0%, 36개월, 자유적립, 비과세(이자소득세+농특세), 월 50만/연 600만 한도.
- 정부기여금: 일반형 6% / 우대형 12% (만기·특별중도해지 시 지급, 외국인 미지급).
- 공통우대(전 기관): 소득플러스 0.5%p(총급여 3,600↓·종합 2,600↓) + 청년재무상담 0.2%p.
- 연계가입(갈아타기): 도약 특별중도해지 사유 인정, 허용 2026.06.22~08.07, 가입신청 ~07.03.
- 그룹: 8%(grp3, +3.0%p) = KB·신한·하나·우리·NH·IBK·우체국 / 7%(grp2, +2.0%p) = 수협·iM·부산·광주·전북·경남·카카오.

## 미반영(정직 표기, YAGNI)

- **신한 연계가입 특별우대 +1.0%p**(2026.8.31까지 **신한** 도약 보유자만 한시): 사용자의 도약 보유은행을 입력받지 않으므로 모델 미반영, `[R]`. 기존 신한 도약 보유자는 실제 최고 8% 도달 가능.
- NH 마이데이터 0.3%p, IBK 청약0.5/중기0.3, 광주 정기예금0.3 등 **2축(급여·카드) 밖 항목**은 모델 미반영(상한 캡으로 영향 제한). 정확 우대는 포털·상품설명서 참조.
- 일부 항목의 충족 횟수·금액(18회/24개월 등)은 모델이 임계로 판정하지 않음(사용자 거래 체크박스로 근사).
