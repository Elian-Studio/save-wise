# save-wise

청년도약계좌 ↔ 청년미래적금 **갈아타기 계산기** (인터랙티브 웹).

두 상품은 만기·월한도·금리·정부기여금이 모두 달라 단순 만기금액 비교가 불공정하다.
이 계산기는 *유지(Stay)* 와 *전환(Switch)* 경로의 순이익·실효수익률을 같은 기준으로 비교하고,
상황(목돈 최대화 / 유동성 중시)에 맞는 추천을 근거와 함께 제시한다.

## 상태

- 🚧 개발 전 단계 — 계산 Spec 초안만 존재 (`claudedocs/calc-spec-draft.md`)
- 일부 계산 규칙(`[R]` 표시)은 deep-research 결과로 확정 예정
- 완료 시 **Vercel** 배포 예정
