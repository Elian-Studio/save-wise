## 0. 비교 프레이밍 (정확성의 핵심)

두 상품은 **만기·월한도·금리·기여금이 모두 다르므로** 단순 만기금액 비교는 불공정하다. 다음 3축으로 본다.

1. **각 경로의 자기 만기 기준 순이익·실효수익률**
   - 유지(Stay): 도약계좌를 도약 가입일+60개월까지 유지.
   - 전환(Switch): 지금 도약 해지 → 해지환급금 확보 + 미래적금 가입일+36개월까지.
2. **공통 시점(예: 지금부터 36개월) 자산가치 비교** — 유동성·기간 차이를 흡수.
3. **유동성/리스크 정성 비교** — 잔여 의무기간, 중도해지 위험, 월 현금부담.

추천 결론은 (1)의 순이익 + (3)의 상황가중으로 산출하고, 비교표로 근거를 노출한다.

## 2. 적금 이자 계산식 (자유적립·단리 가정)

월 정액 `m`만원을 `n`개월 납입, 연금리 `r`(소수):
```
납입원금  P = m × n
세전이자  I = m × r × ( n(n+1)/2 ) / 12
세금      청년도약·미래 모두 비과세 → 세후이자 = I  (이자소득세 15.4% 면제)
```
※ 변동 납입은 회차별 합산으로 일반화하되, 계산기 기본은 정액 가정. 자유적립식 표기.

## 3. 정부기여금

### 미래적금 [R로 정확화]
- 일반형 6% / 우대형 12% — **무엇의 비율인지 확정 필요** (가설: 납입액 대비 매칭, 월 또는 연).
- 잠정 모델: `기여금 = 납입액 × 기여율`. 한도·지급시점 [R].

### 도약계좌 [R로 정확화]
- 총급여 구간별 매칭한도(만원/월) × 매칭비율(%) = 월 기여금. 구간표 [R].
- 유지 시 잔여기간 동안 계속 적립. 전환 시 중단.

## 4. 경로별 만기/해지 가치

### 유지(Stay) — 도약계좌 만기(60개월)
```
잔여개월 rem = 60 - iya_months
최종 = (기납입원금) + (잔여기간 추가납입)
     + 도약이자(전체 60개월, iya_rate)
     + 정부기여금(전체 60개월 누계)
```

### 전환(Switch)
```
[1] 도약 해지환급금 = 기납입원금 + 중도해지이자 ± 정부기여금 정산   ← C2 [R] 가장 중요
[2] 미래적금 만기(36개월) = 납입원금 + 미래이자(bank_rate) + 미래기여금(36개월)
전환 총가치 = [1] + [2]
```
> **C2 미확정**: 갈아타기 해지 시 도약 정부기여금/이자/비과세가 (a) 특례 보존 (b) 일반 중도해지로 일부·전부 상실 중 무엇인지에 따라 [1]이 크게 달라진다. → deep-research C 결과로 확정.
> 신용점수 가점은 도약 기간·납입액 합산 승계되는 것으로 보도 확인됨.

## 5. 은행별 적용금리

```
적용금리 = 기본금리 + Σ(충족한 우대조건 가산)   (은행별 표 [R])
은행 랭킹 = 적용금리 내림차순 → 미래적금 만기이자 최대 은행 추천
```

## 6. 추천 분기 규칙 (초안, [R]로 임계값 보정)

- **전환 유리 신호**: 우대형(12%) 자격 + 미래적금 적용금리 ≥ 도약 현재금리 + 도약 경과 짧음(잔여 김) + 유동성 선호(3년 < 5년) + 도약 금리 하락폭 큼.
- **유지 유리 신호**: 도약 경과 많음(만기 임박, 곧 받을 기여금/이자 큼) + 일반형(6%)만 가능 + 중도해지 시 기여금/이자 상실 큼(C2가 (b)일 때).
- 출력: "전환 시 +X만원 / 유지 시 +Y만원, 차액 Z만원" + 한 줄 결론 + 근거 3가지.

## 7. 시나리오 분기표 (행: 조건 / 열: 유지·전환 순이익)
